### Ajax / Axios

1.引用Axios的js文件
	<scrpit src = "js/axios-0.18.0.js"></script>

2.使用Axios发送请求，并获取相应结果
	axios({
		method:"get",

})

需要提前将官方的Axios的js文件放到js中

![](./javaScript/img/Ajax/概念和作用.png)
![](./javaScript/img/Ajax/同步和异步的区别.png)

Axios 入门摘录：

```html
<script src="js/axios-0.18.0.js"></script>
```

```js
axios({
  method: "get"
})
```
